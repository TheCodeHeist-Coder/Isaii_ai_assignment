# leave page
You are a senior UI/UX designer specializing in enterprise SaaS applications and HR management systems.

Design a modern Leave Management dashboard page for an HRMS platform called "CorporateHR". The interface should feel clean, organized, enterprise-grade, and optimized for reviewing employee leave requests efficiently.

TASK:
Create a responsive HR Leave Management dashboard with analytics cards, leave request tables, filtering tools, and approval actions.

OVERALL DESIGN STYLE:
- Enterprise SaaS UI
- Clean white/light-gray layout
- Deep navy blue primary color
- Rounded cards and containers
- Soft shadows and subtle borders
- Spacious layout with excellent readability
- Minimal distractions and professional hierarchy

PAGE STRUCTURE:

1. LEFT SIDEBAR NAVIGATION
Design a fixed vertical sidebar with:
- Dark charcoal/navy background
- Brand section:
  - "CorporateHR"
  - Subtitle: "Admin Console"

Navigation menu items:
- Dashboard
- Employees
- Attendance
- Leave (active/highlighted)
- Settings

SIDEBAR REQUIREMENTS:
- Minimal line icons
- Active state highlighted in deep blue
- Smooth hover effects
- Bottom user profile card:
  - Avatar image or initials
  - Name: "James Wilson"
  - Role: "HR Director"

2. TOP HEADER
Include:
- Portal title: "HRMS Portal"
- Search bar with placeholder:
  "Search leave requests..."
- Notification bell icon
- Help/support icon
- User avatar on the right

HEADER STYLE:
- Sticky top navigation
- White background
- Thin bottom border
- Spacious alignment

3. PAGE TITLE SECTION

Main heading:
"Leave Management"

Subtitle:
"Review and manage organization-wide leave applications."

Right side:
Primary CTA button:
"+ Request Leave"

BUTTON STYLE:
- Navy blue background
- White text
- Rounded corners
- Soft shadow
- Hover elevation animation

CARD 4:

CARD DESIGN:
- White card background
- Rounded corners
- Soft borders
- Minimal enterprise styling
- Large bold metric numbers
- Small colored status text/icons

TYPOGRAPHY:
- Modern sans-serif font
- Bold page headings
- Medium-weight labels
- Clean table typography
- High readability

SPACING SYSTEM:
- 12px–24px spacing scale
- Large breathing space between sections
- Consistent padding inside cards and tables

MICROINTERACTIONS:
- Hover elevation on cards
- Smooth button transitions
- Row highlight on hover
- Animated status pills
- Sidebar hover effects

RESPONSIVE REQUIREMENTS:
- Fully responsive desktop/tablet/mobile
- Sidebar collapses into drawer on mobile
- Analytics cards stack vertically on smaller screens
- Table becomes horizontally scrollable on mobile
- Header controls rearrange cleanly for smaller devices

ACCESSIBILITY:
- High contrast text
- Keyboard-accessible buttons
- Visible focus states
- Readable font sizing
- Clear status color differentiation
