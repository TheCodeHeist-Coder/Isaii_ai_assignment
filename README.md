# Login Page

You are a senior UI/UX designer and frontend web designer.

Design a modern, clean, enterprise-style HR management system login page inspired by premium SaaS dashboards. The page should look professional, secure, and minimal, with a soft corporate aesthetic.

TASK:
Create a responsive login page for a platform named "CorporateHR".

LAYOUT REQUIREMENTS:
- Centered login card with subtle shadow and rounded corners
- Blurred office background image with light gray/white overlay
- Company logo icon at the top center
- Large heading: "CorporateHR"
- Subtitle: "Human Resources Management System"

LOGIN CARD CONTENT:
1. Two top tabs:
   - Admin Console
   - Employee Portal
   Use pill-style segmented tabs with the active tab highlighted in dark blue.

2. Email input field:
   - Label: "Username or Business Email"
   - Placeholder: "e.g. john.doe@corporate.com"
   - Include mail icon inside input

3. Password field:
   - Label: "Password"
   - "Forgot Password?" link aligned right
   - Password visibility toggle icon inside input

4. Remember me checkbox:
   - Text: "Keep me signed in on this device"

5. Primary CTA button:
   - Full width
   - Dark navy blue background
   - White text
   - Label: "Sign In →"
   - Large rounded corners with hover animation

6. Security information box:
   - Small bordered container
   - Shield/security icon
   - Text explaining MFA protection and secure credentials

7. Footer section:
   - Copyright text
   - Links:
     - Privacy Policy
     - Terms of Service
     - Help Desk

STYLE GUIDE:
- Use a soft grayscale background with subtle gradients
- Primary color: deep navy blue (#0F2B8C or similar)
- Typography:
  - Clean sans-serif font
  - Large bold heading
  - Medium-weight labels
- Use soft shadows and glassmorphism-inspired effects
- Plenty of white space and balanced spacing
- Inputs should have light borders and smooth focus states
- Add subtle hover effects and transitions
- Maintain enterprise SaaS visual quality

RESPONSIVE REQUIREMENTS:
- Fully responsive for desktop, tablet, and mobile
- Login card should remain centered on all screen sizes
- Inputs and button stack cleanly on smaller devices

OUTPUT FORMAT:
Provide:
1. Full UI design description
2. Color palette
3. Typography recommendations
4. Spacing and layout system
5. Component styling details
6. Responsive behavior notes
7. Optional animation/microinteraction suggestions

Do NOT:
- Use overly bright colors
- Add unnecessary graphics
- Make the design look gaming-oriented or futuristic
- Overcrowd the interface
